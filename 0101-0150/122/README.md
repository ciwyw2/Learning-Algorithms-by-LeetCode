+ 贪心算法：在对问题求解时，总是做出在当前看来最好的选择。
+ 即不从整体上最优加以考虑，只求局部最优解。
+ 对这个问题获得可能的最大利润的手段为：买入后的次日，只要股价上涨就卖出，如果后一天股价下跌则前一天不买入。
+ 后一条如果没有时间机器任谁也无法事先知道。
+ 所以下面解法只能用于在 n 天中每天股价都已知的情况下求个理想值，并不能用于指导实际。
